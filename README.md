# 织里童装预测 SaaS 演示

这是一个面向湖州织里童装档口供应商的供应商端 SaaS 静态演示原型，用于展示未来 7/14/30 天爆款预测、款式图片、预测维度、原材料建议、备货建议，以及商详和短视频内容制作入口。

演示地址：

https://rowesy0801.github.io/zhili-saas-demo/

## 产品约束

- 供应商端只展示预测结论、预测维度和可执行建议。
- 不展示底层具体经营数据，例如具体成交、平台排行、搜索规模或模型内部指标。
- 第一版聚焦湖州织里童装，后续可扩展到常熟、广州等服饰产业带。

## 页面能力

- 未来 7 天、14 天、30 天预测周期切换。
- 推荐款式卡片和右侧款式详情联动。
- 展示预测等级、适合年龄段、建议上新时间、核心预测维度。
- 展示原材料建议和备货建议。
- 提供 WEB 商详、APP 商详和短视频脚本生成入口。

## 本地预览

直接用浏览器打开 `index.html` 即可预览。
