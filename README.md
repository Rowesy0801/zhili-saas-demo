# 织里童装爆款预测 SaaS 演示

这是面向湖州织里童装档口供应商的前端交互 demo，用于客户演示“款式预测 + 商详图片风格预测 + AI 内容生成 + 内部后台运营管理”的完整闭环。

演示地址：

https://rowesy0801.github.io/zhili-saas-demo/

## 本次版本能力

- 供应商端 SaaS：预测首页、爆款款式库、原材料建议、商详制作、短视频制作、AI 定制、订阅额度。
- 内部运营后台：运营总览、商家管理、入驻审核、订阅订单、Token 管理、转化漏斗、内容工单、数据模型监控、平台销量导入。
- 商详制作升级为 AI 商详图片工作台：
  - 支持从预测款式进入生成。
  - 支持手动上传多张商品照片。
  - 支持多套商详图片风格选择。
  - 支持生成商详图片提示词和视频生成提示词。
  - 支持员工通过对话框对图片细节进行微调。
- 后端数据暂用前端测试数据模拟，适合客户演示和需求评审。

## 产品约束

- 供应商端只展示预测结论、预测维度、原材料建议和备货动作。
- 供应商端不展示底层具体平台数据、具体排行或模型内部指标。
- 内部后台可管理平台数据导入、模型监控、风格学习、提示词版本和内容工单。

## 本地预览

直接用浏览器打开 `index.html` 即可预览，也可以用任意静态文件服务运行本目录。
