# 智衣预测 SaaS 演示

这是面向湖州织里童装和常熟服装产业带供应商的前端交互 demo，用于演示“爆款预测 + 高转化商详风格预测 + 商详套图/短视频生成 + 后台数据导入”的完整业务闭环。

演示地址：
https://rowesy0801.github.io/zhili-saas-demo/

## 本版能力

- 用户端固定导航：预测首页、爆款款式库、原材料建议、商详制作、短视频制作、AI定制、订阅额度。
- 后台固定导航：运营总览、商家管理、入驻审核、订阅订单、Token管理、转化漏斗、内容工单、数据模型监控、平台销量导入。
- 新增常熟服装产业带，覆盖女装、男装、老年装；湖州织里保留童装。
- 用户端只查看预测款式、预测维度、原材料建议和备货建议。
- 用户端支持从预测款式进入商详制作，也支持模拟上传多张商品图。
- 商详制作支持多套风格选择、提示词编辑、对话式微调、商详套图生成、短视频分镜生成和导出。
- 积分消耗规则：商详图片 2 积分/张，短视频 20 积分/段；生成前展示预计消耗，充值和使用记录可查看。
- 后台支持平台销售数据、高转化商详图片、短视频样本、产业带/类目配置的导入演示。

## 产品约束

- 进入系统的数据统一由后台管理导入。
- 供应商端只给出结论、预测维度、材料建议和生成入口。
- 供应商端不展示底层平台明细、具体排行或模型内部指标。
- 内部后台可以查看导入、校验、模型监控、提示词版本、积分流水和内容工单。

## 本地预览

直接用浏览器打开 `index.html` 即可预览，也可以用任意静态文件服务运行本目录。
